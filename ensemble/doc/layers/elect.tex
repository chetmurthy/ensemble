\begin{Layer}{ELECT}

This layer implements a leader election protocol.  It determines when a member
should become the coordinator.  Election is done by delivering an \Dn{Elect}
event at the new coordinator.

\begin{Protocol}
When a member suspects all lower ranked members of being faulty, that member elects
itself as coordinator.
\end{Protocol}

\begin{Parameters}
\item None
\end{Parameters}

\begin{Properties}
\item
\Up{Suspect} events may cause a \Dn{Elect} event to be generated.
\end{Properties}

\end{Sources}

\end{Layer}
